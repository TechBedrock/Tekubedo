## 🛠 本地开发

如果你想参与编写或修改 Wiki，可以在本地运行：

```bash
# 克隆仓库
git clone https://github.com/TechBedrock/Tekubedo.git
cd Tekubedo

# 安装依赖
npm install

# 启动本地开发服务器
npm run start
```

打开浏览器访问 <http://localhost:3000>，即可预览。
